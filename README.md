# Große Belegarbeit
**Thema**: _Multimodales Intraoperatives Neuroimaging_

**Aufgabenstellung**:

In der Klinik und Poliklinik für Neurochirurgie des Universitätsklinikums Dresden werden intraoperativ 2D Neuroimaging Techniken (bspw. Optische Bildgebung, Thermografie) zur Abgrenzung von eloquenten Arealen (Sprachzentrum, sensorischer Kortex) und Tumorgewebe eingesetzt. Zur Verbesserung der Genauigkeit der statistischen Datenanalyse sowie zur anatomischen Validierung werden diese 2D Bilddaten mittels Bildregistrierung und Parallelprojektion mit präoperativen 3D MRT-Datensätzen fusioniert. Im Rahmen dieser Arbeit sollen folgende Aspekte ergänzt werden:
- Literaturrecherche zu generischen Ansätzen der multimodalen Bildfusion
- Automatische Kalibrierung des Kamera-Setups zur -Referenzeinheit
- Bildfusion per projektiver Abbildung in 3D Slicer
- Evaluation von Einflussfaktoren auf die Genauigkeit und Geschwindigkeit des Frameworks
